## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# prisma genereted
$ npx prisma genereted

# init database BEFORE seed
$ npx prisma migrate dev

# seeding database BEFORE start
$ npm run seed

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
